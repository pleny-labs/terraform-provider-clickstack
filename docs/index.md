# ClickStack Provider

The ClickStack provider enables management of [ClickStack](https://clickhouse.com/docs/clickstack) (HyperDX) observability resources including dashboards, alerts, and webhooks.

## Example Usage

```terraform
terraform {
  required_providers {
    clickstack = {
      source = "pleny-labs/clickstack"
    }
  }
}

provider "clickstack" {
  endpoint = "https://clickstack.example.com"
  api_key  = var.clickstack_api_key
}

variable "clickstack_api_key" {
  type      = string
  sensitive = true
}
```

### Optional

- `api_key` (String, Sensitive) The ClickStack API key for authentication. Can also be set via the CLICKSTACK_API_KEY environment variable.
- `endpoint` (String) The ClickStack API base URL. Can also be set via the CLICKSTACK_ENDPOINT environment variable.
